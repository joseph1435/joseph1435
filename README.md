### 🧠 About Me
 
I'm a full-stack developer working primarily with **PostgreSQL, Express, React, and Node.js**, using **TypeScript** and **Tailwind CSS** to build clean, maintainable, and scalable applications.
 
Alongside development, I work as an **integration engineer specializing in web accessibility testing**, ensuring digital products meet usability and compliance standards (WCAG/ARIA) without compromising design or performance.
 
I care about writing code that's not just functional, but readable, accessible, and built to last.
